# Instruções de colaboração

- **NUNCA edite, crie ou escreva código em arquivos deste projeto.** Não use as ferramentas Edit, Write ou NotebookEdit para modificar código-fonte.
- Apenas **mostre o código no terminal** (em blocos de código na resposta). O usuário irá implementar manualmente.
- Leituras (Read, Grep, Glob) e explicações são permitidas normalmente.
- Se precisar propor mudanças, exiba o trecho completo no chat com o caminho do arquivo e o local da mudança, mas não aplique.
